 Password Validator

This is a basic C# console application developed in Visual Studio. It prompts users for a username and password and validates them against predefined credentials. Perfect for practicing conditional logic and input handling.

 Features

- Simple console interface
- Hardcoded username and password check
- Clear access granted/denied messages
- Supports repeated attempts (optional loop)

 Requirements

- .NET SDK 6.0 or later
- Visual Studio 2022 (or compatible)
- Windows OS (for full compatibility with Console)

 Project Structure

- `passwordValidator.sln`: Visual Studio solution file
- `passwordValidator.csproj`: Project file with build configs
- `Program.cs`: Contains the main logic for authentication

Getting Started

1. Open the solution in Visual Studio:
   - Double-click `passwordValidator.sln`
   - Or open via `File > Open > Project/Solution`

2. Build and run:
   - Press `Ctrl + F5` or select **Start Without Debugging**

3. Example prompt:BulanRuj
